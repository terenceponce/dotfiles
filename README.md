# Terence's Developer Setup

This repo contains instructions on how to setup a development environment from scratch.

Of course, this is all based on my own personal preferences.

## Table of Contents

- [Terminal Setup](#terminal-setup)

## Terminal Setup

You can use whatever terminal app you like as long as it's not ancient.

For Mac OS X, that means you will almost always go for [iTerm](https://iterm2.com) if you want a good experience. You can also just use the default terminal app, but YMMV.

For Linux, you can use whatever your desktop environment is providing out of the box.

### Terminal Colors

Most terminal apps already have this, but if it doesn't, find out how to install [Solarized](https://ethanschoonover.com/solarized/) Dark and set that as the default color preset.

### Terminal Font

Set [FiraCode](https://github.com/tonsky/FiraCode) as the default font for your terminal. It's the most popular font for programmers right now because it looks good and it supports a lot of ligatures.
