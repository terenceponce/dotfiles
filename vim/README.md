# Vim Configuration

This works for both [Vim](https://www.vim.org/) and [Neovim](https://neovim.io/)

## Quick Start

1. Install Neovim

```
brew install neovim
```

2. Symlink this folder to the locations where Vim/Neovim will use them

```
make link
```

3. Open Neovim

```
nvim
```

4. Install Vim Plugins

```
:PlugInstall
```
